# 🎓 CurricuForge AI

## What is this?

CurricuForge AI is a **Streamlit-based web application** that uses **Groq (Llama 3.3-70B)** to generate and optimize academic curricula.  
It helps users design course structures, recommend topics, plan full programs, map learning outcomes, and improve academic quality using AI.

---

## File Information

**File Name:** app.py  

**Description:**  
Main application file that:
- Builds the UI using Streamlit  
- Takes user input for curriculum design  
- Sends prompts to Groq API  
- Displays AI-generated results  

---

## Key Functionalities

- Course Structure Generator  
- Topic Recommendation Engine  
- Curriculum Planner  
- Learning Outcome Mapping (OBE)  
- Academic Optimization  
- AI Chat Assistant  

---

## Technologies Used

- Python  
- Streamlit  
- Groq API (Llama 3.3-70B)
